## Quick Commands

```bash
# Diagnostic
npm run diagnose

# Create admin user
npm run create-admin

# Start backend
npm run dev

# Start frontend
npm run dev

# Test health check
curl http://localhost:5002/api/health

# Test login
curl -X POST http://localhost:5002/api/auth/login \
  -H "Content-Type: application/json" \
  -d "{\"email\":\"admin@draftin.com\",\"password\":\"admin123\"}"
```
