📌 LANGKAH CEPAT:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  RESTART BACKEND
   cd backend
   npm start
   
   ✅ Harus muncul: "Server running on http://localhost:5002"

2️⃣  JALANKAN FRONTEND (Terminal Baru)
   cd frontend
   npm run dev
   
   ✅ Harus muncul: "Local: http://localhost:5173/"

3️⃣  BUKA BROWSER
   http://localhost:5173/

4️⃣  LOGIN SEBAGAI CUSTOMER
   - Gunakan akun customer (bukan admin)
   - Atau register akun baru

5️⃣  TAMBAH PRODUK KE CART
   - Browse produk
   - Klik "Add to Cart"

6️⃣  CHECKOUT
   - Klik icon Cart
   - Klik "Checkout"
   - Popup Midtrans akan muncul

7️⃣  BAYAR
   - Pilih metode pembayaran
   - Selesaikan pembayaran
   - CATAT Order ID!

8️⃣  CEK STATUS
   - Buka: https://dashboard.midtrans.com/
   - Menu "Transactions"
   - Cari transaksi Anda
   - Status harus "Settlement" jika berhasil

9️⃣  REFRESH APLIKASI
   - Kembali ke aplikasi
   - Refresh halaman (F5)
   - Status order akan update

❌ Popup Midtrans tidak muncul
   → Cek console browser (F12)
   → Pastikan backend jalan
   → Cek API keys di .env

❌ Status tidak update
   → Refresh halaman (F5)
   → Cek dashboard Midtrans
   → Logout-login lagi

❌ Pembayaran gagal
   → Coba kartu lain
   → Coba metode pembayaran lain
   → Cek saldo/limit kartu
