1. RESTART BACKEND SERVER
   cd backend
   npm start

2. TEST TRANSAKSI KECIL
   - Buat order dengan produk murah (Rp 1.000 - 10.000)
   - Gunakan kartu asli untuk test
   - Verifikasi pembayaran masuk

3. SETUP WEBHOOK (WAJIB!)
   - Login ke dashboard.midtrans.com
   - Settings → Configuration
   - Set Payment Notification URL
   - Untuk localhost: gunakan ngrok

4. MONITOR TRANSAKSI
   - Cek dashboard.midtrans.com/transactions
   - Pantau status pembayaran
   - Verifikasi settlement
